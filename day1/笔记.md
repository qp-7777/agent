# SSE — Server-Sent Events（服务器推送事件）

## 概述

SSE 是一种**服务器主动推送**技术，前端只需建立连接，服务端持续发送数据。

- 单向长连接：**服务端 → 前端**，前端不能发送消息
- 基于 HTTP/HTTPS，无需额外协议（80/443 端口即可），无需握手升级
- **自动断线重连**，无需手动处理

---

## SSE 格式规范

### 1. 响应头（后端必须设置）

```http
Content-Type: text/event-stream
Cache-Control: no-cache
Connection: keep-alive
```

> 三个缺一不可，少一个前端都无法正确解析。

### 2. 消息格式三要素

| 字段 | 说明 |
|------|------|
| `data: xxx` | 真正传输的数据（**必填**） |
| `event: xxx` | 自定义事件名（可选，区分不同消息类型） |
| `retry: xxx` | 断开后多久（毫秒）自动重连 |

**每条消息必须以两个换行 `\n\n` 结尾**，表示一条消息结束。

### 3. 结束标识

大模型流式结尾返回 `data: [DONE]`，代表传输完成，前端收到后关闭连接。

```text
data: [DONE]

```

## 服务端示例（Node.js + Express）

```js
app.get('/stream', (req, res) => {
    // 1. 设置 SSE 响应头
    res.setHeader('Content-Type', 'text/event-stream');
    res.setHeader('Cache-Control', 'no-cache');
    res.setHeader('Connection', 'keep-alive');

    // 2. 发送自定义事件（连接建立后立即推送）
    res.write('event: msg\ndata: 欢迎连接\n\n');

    // 3. 定时发送默认消息
    let count = 0;
    const timer = setInterval(() => {
        count++;
        res.write(`data: 流式输出第${count}条消息\n\n`);

        if (count >= 10) {
            res.write(`data: [DONE]\n\n`);
            clearInterval(timer);
            res.end();
        }
    }, 500);

    // 4. 前端断开连接时清理资源，防止内存泄漏
    req.on('close', () => {
        clearInterval(timer);
    });
});
```

## 客户端示例（浏览器）

```js
// 1. 建立连接
const sse = new EventSource('http://localhost:3000/stream');

// 2. 监听默认消息
sse.onmessage = (e) => {
    if (e.data === '[DONE]') {
        console.log('传输完成');
        sse.close();
        return;
    }
    console.log('收到消息:', e.data);
};

// 3. 监听自定义事件
sse.addEventListener('msg', (e) => {
    console.log('自定义事件:', e.data);
});

// 4. 监听连接建立
sse.onopen = () => {
    console.log('连接已建立');
};

// 5. 监听错误（断网、接口404等）
sse.onerror = (err) => {
    console.error('连接异常，自动重试中...', err);
};

// 6. 手动关闭连接
sse.close();
```

## 发送 JSON 数据的推荐方式

服务端发送 `JSON.stringify(...)`，前端用 `JSON.parse(...)` 解析：

```js
// 服务端
const data = JSON.stringify({ type: 'notification', message: '欢迎连接', time: Date.now() });
res.write(`event: msg\ndata: ${data}\n\n`);

// 前端
sse.addEventListener('msg', (e) => {
    const data = JSON.parse(e.data);
    console.log(data.type, data.message);
});
```
